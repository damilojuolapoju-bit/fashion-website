# Lumière Couture - Fashion House Website

## 📋 Project Overview

**Lumière Couture** is a comprehensive, modern fashion house website built with HTML, CSS, and JavaScript. This project showcases a complete e-commerce and event management platform for an exclusive fashion brand.

### Fashion House Name
**Lumière Couture** - "Elegance Redefined"
- Meaning: "Light Tailoring" in French, representing sophistication and innovation
- Vision: A luxury fashion brand combining artistic design with modern aesthetics

---

## ✨ Key Features Implemented

### 1. **5+ Web Pages**
- ✅ **Home (index.html)** - Landing page with hero section and featured products
- ✅ **Products (products.html)** - Complete product showcase with filtering
- ✅ **Board of Trustees (board.html)** - Team member profiles with interactive hover effects
- ✅ **Events (events.html)** - Upcoming events timeline and registration
- ✅ **Contact & Appointments (inquiries.html)** - Inquiry forms and appointment booking system

### 2. **Product Showcase**
- Dynamic product grid with 9+ products
- Categories: Dresses, Suits, and Accessories
- Real-time filtering functionality
- Shopping cart system with local storage
- Price display and product descriptions
- Add to cart functionality with notifications

### 3. **Consistent Link Structure**
- Unified navigation menu across all pages
- Sticky header with active page highlighting
- Footer with quick links on every page
- Consistent URL structure (index.html, products.html, board.html, events.html, inquiries.html)

### 4. **Board of Trustees**
- 6 team members with professional roles
- Avatar emojis representing each member
- Interactive hover effects revealing bio and contact info
- Values section highlighting team principles
- Smooth animations and transitions

### 5. **Inquiries & Appointments**
- Comprehensive inquiry form with:
  - Name, email, phone validation
  - Inquiry type selector
  - Message textarea
  - Newsletter subscription option
- Advanced appointment booking with:
  - Date and time selection
  - Service type selection
  - Additional notes
  - Booking confirmation
- Bookings display section
- Form validation and success messages
- Local storage for persisting bookings

### 6. **Upcoming Events**
- Interactive event timeline with 5 featured events
- Event details: Date, time, location, description
- Individual registration buttons
- Event statistics section
- Responsive event cards with hover effects

### 7. **Marquee Section**
- Sticky marquee at top of every page
- Scrolling announcement text
- Auto-playing animation
- Pause on hover functionality
- Purple gradient background with custom styling
- Displays upcoming events and promotions

### 8. **JavaScript Features**
All pages include interactive JavaScript functionality:
- **Dynamic Navigation**: Active page highlighting
- **Product Filtering**: Filter by category with smooth animations
- **Shopping Cart**: Add/remove items with real-time updates
- **Form Validation**: Email, required fields, and input validation
- **Interactive Components**: 
  - Board member hover effects
  - Event card interactions
  - Notification system
- **Local Storage**: Persist cart and bookings data
- **Smooth Scrolling**: Smooth page navigation
- **DOM Manipulation**: Dynamic content rendering
- **Event Handling**: Form submissions, button clicks, hover effects

### 9. **External CSS**
- **Single stylesheet: `styles.css`** (900+ lines)
- Features:
  - CSS Grid and Flexbox layouts
  - Responsive design (Mobile, Tablet, Desktop)
  - Gradient backgrounds
  - Smooth transitions and animations
  - Hover effects and visual feedback
  - Typography hierarchy
  - Color scheme: Purple (#667eea, #764ba2) and neutral tones
  - Media queries for mobile optimization
